# SyncLink - Novel Discovery Platform 📚

Welcome to **SyncLink**! This is a web application created as a first-year Computer Science project. It allows users to search for thousands of books and novels in real-time.

## 🚀 Features
- **Live Search:** Connects to the OpenLibrary API to find any book.
- **Bookmarks:** Save your favorite titles locally.
- **Dark Mode UI:** A clean, modern interface designed for readers.

## 🛠️ Built With
- **HTML5 & CSS3** (Custom Grid Layouts)
- **Vanilla JavaScript** (Fetch API & LocalStorage)
- **Hosted on GitHub Pages**
